# Netflix-Content-strategy-EDA
Netflix has 222M+ subscribers — but what to produce next?  This Python EDA on 8,807 Netflix titles uncovers trends in  content type, genre, ratings &amp; country distribution. Finds:  TV Shows rising post-2016, India is the top growth market,  Drama &amp; Comedy dominate. Delivers 6 business recommendations.  Data analysis drives real decisions.
